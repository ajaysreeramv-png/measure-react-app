# Measurement Converter

Open `index.html` in the `converter` folder with your browser (double-click or use a local server).

Quick run (PowerShell):

```powershell
cd "C:\Users\ajays\OneDrive\Desktop\measure\converter"
# Open directly:
ii .\index.html
# Or serve locally:
py -3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

This app uses React via CDN and requires no build step.
